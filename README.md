# WinNative CI Releases

This repository is used as the release target for automated WinNative CI builds.

## Purpose

GitHub Actions from the main WinNative repository publish APK builds here so each pull request can have a dedicated release page with attached artifacts.

## What You Will Find Here

- Prerelease entries for active pull requests
- APK artifacts generated by CI
- Release notes linking back to the source pull request

## Notes

- Releases in this repository are generated automatically.
- Assets may be replaced as new commits are pushed to the same pull request.
- The latest APKs for a PR are always attached to that PR's release.

## Source Repository

Source changes are developed in the main WinNative repository and mirrored here as CI release artifacts.
